BRANCH -  lugar onde fica armazenado um arquivo no git; repositorio
branch.master

GIT INIT = criar um repositorio; 1x para cada projeto	
GIT ADD = para adicionar as mudanças
GIT COMMIT = M "ADD ALGUNS ARQUIVOS"; para dar git commit precisa ter add algo, para atualizar as mudanças
GIT STATUS = checar o que mudou
GIT CHECKOUT -B NOME_DO_NOVO_BRANCH = criar um novo branch
GIT CHECKOUT MASTER = para voltar para o branch master
GIT REMOTE ADD ORIGIN LINK DO REPOSITORIO DO GITHUB

1. cd Documents/Projeto
2. git init
3. git add .
4. git commit -m "mensagem"
5. git status 


 git checkout nova_funcionalidade 
 para criar = git checkout -b nome da nova branch
